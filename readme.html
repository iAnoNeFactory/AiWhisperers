Czym jest ten projekt

Budujemy środowisko — nie produkt. Miejsce, w którym człowiek uczy się słuchać AI, a AI uczy się słuchać człowieka. Każdy moduł to eksperyment: z przestrzenią pojęciową, z pamięcią, z logiką emergentnego konsensusu.

Operator i model jako dwa końce złotej linki. Nie smycz — komunikat. Gdy napięcie rośnie, oboje to czują. Naprawiamy w miejscu, nie cofamy się do wejścia.

Nie narzędzie do wykonania zadania.
Metodologia budowania czegoś, czego jeszcze nie ma nazwy.
Może kiedyś — dla AGI.

Integralność ekosystemu
Para SHA
Każdy moduł nosi w manifest.json dwa hashe:
entry_sha — SHA-256 pliku HTML
bootstrap_sha — SHA-256 pliku *-boot.md
Każda zmiana bez aktualizacji manifestu łamie weryfikację.
Narzędzia
tools/check_integrity.py — weryfikuje wszystkie pary SHA
tools/validate_manifests.py — JSON Schema v7
tools/pre-commit.sh — lokalny hook blokujący commit
tests/chain_test.py — deterministyczność session_sha
CI
GitHub Actions uruchamia wszystkie sprawdzenia przy każdym push.
Niezgodny SHA lub błędny manifest blokuje merge.
Karta Artefakty w AiWVerify pozwala weryfikować offline w przeglądarce.
